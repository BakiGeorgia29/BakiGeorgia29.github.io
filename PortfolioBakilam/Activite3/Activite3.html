CSS

Le CSS est un langage de style , il permet de rendre les pages webs plus agréable. L'association du HTML et du CSS permet d'obtenir les pages plus:

  • Expressive
  • Cohérente
  • Equilibrée
  • Professionnelle
  • Adaptif

Il comporte sa synthaxe , ces spécifités comme tout autre langage. Cette page a été constitué du HTML et CSS , à partir de ce bouton vous aller découvrir d'autres pages trés simple , stylisé.
